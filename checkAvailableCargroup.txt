# test if car category is available:

1. sum cars of each category for selected rental office:
A 4
B 3
C 6
D 2

2. sum all reservations that overlap selected period: 
A 4
B 2
C 3
D 2


3. available cars:
A: 4 - 4 = 0
B: 3 - 2 = 1 > 0 => available
C: 6 - 3 = 3 > 0 => available
D: 2 - 2 = 0

=> category B and C is available for selected period
